# SMS2Mail
This is an Android application designed for forward SMS to a specific mailbox via email. 

## Notable Features
- Send SMS to mailbox
- Support SSL, TLS (including STARTTLS) protocol
- Pure compose

## Changelog

- Refactor: Use Compose instead of XML UI Framework
